# NEURAL-NOTES
This is monorepo of `neural-notes` app.

![example](images/examples.png)

## Quick Start
1. install just &  docker

2. in your shell
    ```bash
    git clone git@github.com:lyh4215/neural-notes.git
    cd neural-notes
    just env
    just dev
    ```
3. go to http://localhost:5173
